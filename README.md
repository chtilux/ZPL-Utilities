# ZPL-Utilities
Abattoir Ettelbruck

Installation
To run, the program needs :
•	zplUtilities.exe
•	a sub-folder /x86 with pdfium.dll (32 bits)
•	a sub-folder /xml with zpHelp.xml
•	a sub/folder /labelary with a blank.pdf file
